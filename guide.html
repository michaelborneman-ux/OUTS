1 — Unlocking the App

The app locks itself each day with a PIN to prevent accidental use. Enter the PIN on the lock screen to begin your shift.

4959

Daily unlock PIN

Once unlocked the app stays open for the rest of the day. You will not need to enter the PIN again unless the app is fully closed and reopened the next day.

2 — Home Screen

The home screen shows a summary of your day and lists all loaded bundles.

Summary bar

Shows total Bundles, total Cards (meter addresses), how many have been Read, and how many bundles have been Sent.

Bundle status badges

In Progress — some readings recorded
Complete — all cards read or skipped
Sent — bundle marked as sent

Reader name

Tap the name at the top to change it. The name is printed on the report.

3 — Loading a Route (CSV)

Routes are loaded by importing a CSV file from the home screen. Tap the upload button and select the file from your device.

Tip: Any extra columns in the CSV are preserved and will appear in the exported file unchanged.

4 — Bundle List Page

Tapping a bundle opens its address list. From here you can:

  • See every address in the bundle and their reading status
  • Search or filter addresses (missed / skipped)
  • Tap an address to open its data entry card
  • Optimize the route order
  • Open the bundle report
  • Mark the bundle as Sent

Estimate totals

The header shows estimated counts split by meter size category:

3 Est — meters with size 3
4–6 Est — meters with size 4, 5, or 6
7+ Est — meters with size 7 or larger

5 — Data Entry Page

This is the main screen for recording a meter reading. It shows one address at a time.

Reading field

Enter the digits shown on the meter. The app highlights the field in red if the digit count does not match the expected count for that meter size. The expected digit count by meter size:

UD — 6 digits
UC, UB — 5 digits
S… prefix — 5 digits
C, D, G prefix — 4 digits
All others — no restriction

Saving a reading

  1. Type the meter reading into the Reading field.
  2. Optionally add a Comment.
  3. Tap Save. The date and time are recorded automatically and the app advances to the next card.

Once saved, the reading field is locked to prevent accidental changes. To correct a reading, open the burger menu and tap Delete Reading, then re-enter it.

Skipping a meter

If you cannot read a meter, select a reason from the Skip dropdown (see Skip Reasons) and tap Save. A skipped card counts as complete.

Burger menu options

Reverse Direction

Normally saving advances forward to the next card. Enable this to advance backward — useful when walking back down a street.

Search List

Returns to the bundle list with the search bar focused so you can jump to a specific address.

Report

Opens the bundle report page directly.

Delete Reading locked cards only

Removes a saved reading so it can be re-entered. A confirmation warning is shown before deletion.

6 — Map View

The map shows all geocoded addresses for the current bundle as colour-coded markers.

Marker colours

● Blue/Purple — unread
● Green — read
● Gold — skipped
● Pulsing blue — your current GPS location

Tapping a marker

Taps a marker to see the address. From the popup you can open that card directly or fix the pin location if it is in the wrong place.

Fixing a location

  1. Tap the marker that is in the wrong position.
  2. Tap Fix Location in the popup.
  3. Enter the correct latitude and longitude in decimal format (e.g. 41.8781, -87.6298).
  4. Tap Confirm. The marker moves and the corrected position is saved permanently.

Geocoding

When a bundle is opened, the app geocodes all addresses using OpenStreetMap. Geocoded positions are cached so the map works offline after the first load. Use Geocode All on the bundle page while you have a signal before heading into the field.

Tip: Map tiles are also cached for offline use. Zoom into your work area while online so the tiles are stored.

7 — Report Page

The report page generates a printable summary for a bundle. Open it from the bundle list header button, or from the Report item in the data entry burger menu.

The report includes:

  • Bundle name, date, and reader name
  • Meter counts by estimate category (3-day, 4–6 day, 7+ day)
  • Total estimated amount
  • Signature line

Use the Print button to print or save as PDF.

Tip: Mark a bundle as Sent after submitting it. Sent bundles are tracked on the totals screen so you have a record of what has been turned in.

8 — Skip Reasons

When a meter cannot be read, select the appropriate reason before saving. The skip reason is exported with the CSV.

  • Meter Removed
  • Condensation
  • Unsafe Condition
  • Meter Under Porch
  • Meter Damaged
  • Special Arrangement
  • Dead Battery
  • Other (free text)
Other: Selecting "Other" reveals a text field — type a custom reason before saving.

9 — Route Optimization

The app can reorder the addresses in a bundle to create an efficient route.

  1. Open a bundle and tap Optimize Route.
  2. The list enters pick-start mode. Tap the address you want to start from.
  3. The app calculates the shortest path from that point using road distances (online) or straight-line distances (offline).
  4. The list reorders and sequence numbers update.
Reset: Tap Reset Order to restore the original CSV order at any time.
Offline use: If you have no signal, optimization still works using straight-line (as-the-crow-flies) distances. Geocode all addresses while online beforehand for the best results.

10 — Tips & Notes

Works offline

All data is saved locally on the device. You do not need a signal to record readings. Sync and export when you are back in range.

Exporting data

Use the Export CSV button on the bundle list to download a completed route file. Column names are converted to the standard output format automatically.

Crash recovery

Readings are saved to a local backup after every entry. If the app closes unexpectedly, your data will be restored the next time you open it.

Multiple units at one address

When multiple meters share the same address, the map shows a picker so you can select the correct unit before opening its card.

⚠ Do not clear your browser cache

All readings, bundle data, geocache, and settings are stored in the browser. Clearing your browser cache or site data will permanently delete everything that has not been exported or backed up. Always export your CSV or confirm your auto-backup folder is set before clearing any browser data.

Install as an app

On iOS tap Share → Add to Home Screen. On Android tap Menu → Add to Home Screen. This gives you a full-screen experience with no browser bar.
